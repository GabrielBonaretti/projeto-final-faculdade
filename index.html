Our Goal!

The RetroGame is an innovative project aimed at providing a unique e-commerce experience, centered around the sale of vintage games. Developed as part of the HTML, CSS, Javascript, and PHP course, this project is more than just a web application; it's a celebration of digital nostalgia and an opportunity to apply technical skills acquired throughout the course.

Key Technical Features?

  • We utilized HTML to create a responsive and well-organized web page structure. Each element has been carefully designed to ensure an intuitive and enjoyable browsing experience.

  • The retro aesthetic is at the heart of RetroGame Emporium, and CSS plays a crucial role in creating that atmosphere. We implemented styles that pay homage to pixelated graphics and color palettes characteristic of classic games.

  • The application of Javascript adds interactive layers to the project. From dynamic filtering and sorting features to real-time feedback elements, Javascript enriches the user experience and makes navigation more efficient.

  • Integrating security techniques ensures user information is protected. Additionally, dynamic forms were implemented to facilitate the purchasing process, providing a friendly and intuitive interaction
